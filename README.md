# Cyclistic Bike Share Case Study

## Project Overview
This project was completed as part of the Google Data Analytics Professional Certificate Capstone.

The objective of this case study was to analyze Cyclistic bike-share data to identify differences between annual members and casual riders, and to develop data-driven recommendations for converting more casual riders into annual members.

---

## Tools Used
- Microsoft Excel
- Tableau Public

---

## Project Tasks
- Data cleaning and preparation
- Ride length calculation
- Descriptive statistics
- Pivot table analysis
- Data visualization
- Business recommendations

---

## Key Findings
- Annual members generated more rides than casual riders.
- Casual riders had longer average ride durations.
- Weekend rides were dominated by casual riders.
- Weekday rides were dominated by annual members.
- Saturday recorded the highest number of rides.

---

## Business Recommendations
- Introduce promotional discounts and referral incentives for annual memberships.
- Conduct surveys to better understand casual rider behavior.
- Provide personalized yearly ride summaries showing potential savings from becoming an annual member.
